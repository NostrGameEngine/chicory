# Chicory | NGE

This is a fork of [chicory](https://github.com/dylibso/chicory) (a JVM native WebAssembly runtime) patched to be used in [Nostr Game Engine](https://ngengine.org). The patches are mostly related to making the runtime compilable with TeaVM.

If you are looking for a wasm runtime for your project unrelated to NGE, please check out the original [dylibso/chicory](https://github.com/dylibso/chicory).
